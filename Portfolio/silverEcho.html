Silver Echo is a stealth based puzzle game created in under 36 hours for the Global Game Jam 2017 at RIT.
The goal of the game is to avoid the robots using your sonar vision and collect all 3 keys to esacepe your mansion.

Role

  • Game Producer: Using the scrum ideology managed a team of 8 to ensure all the goals and tasks were met in a timely manner.
  • UI/UX Developer: Developed the UI and HUD, as well as the Sound Manager. Designed a dynamic color picker for the enemies and the hero.

Tools Used

  • Unity 5.5/C#
  • Visual Studio 2015
  • Trello
  • Photoshop CC
  • Autodesk Maya 2016
  • Github
